# StreamTimer
  Simple AHK script displaying a 10s looping countdown in the top-right corner. Useful for accurately measuring streaming latency in complex gaming environments. Press Esc to quit. (Requires Windowed or Borderless Window mode.)
  
  简单的ahk脚本，打开后会在屏幕右上角显示一个满10秒归零的倒计时，可以比较准确地测试在游戏画面复杂时串流的屏摄延迟，Esc退出。（游戏需要以窗口化或无边框窗口运行）
